Le vocabulaire

Archive

Une archive est un fichier contenant plusieurs autres fichiers, sous forme d'arborescence ou non. L'archive peut être compressée et/ou cryptée. Les archives peuvent être utilisé par tous les utilisateurs d'un ordinateur sous la condition d'avoir un logiciel adapté. De nombreux logiciels existent, voici une liste non-exaustive des logiciels les plus répandu :

  • WinRar, logiciel Windows gérant les archives de différents types.
  • WinZip, logiciel Windows gérant les archives surtout de type *.zip.
  • 7zip, logiciel multiplate-forme gérant tous les types répandus d'archive.
  • tar, logiciel en ligne de commande Linux pour la gestion d'archive de tous types.
  • L'explorateur de fichier Windows permet de réaliser certaines actions sur les archives.

Les archives peuvent se présenter sous plusieurs formats, ne servant pas tous à la même chose. Les plus répandu sont :

  • *.rar
  • *.zip
  • *.tar.gz
  • *.jar, Fichier compressé servant d’exécutable aux machines virtuelles JAVA
  • *.iso, Fichier compressé d'une image de disque servant à la sauvegarde ou à l’installation de programme
  • *.deb, Fichier archive contenant les informations et les données pour l’installation d'un logiciel d'un système basé sous Debian

L'archive permet de déplacer, de sauvegarder ou de protéger des données facilement grâce à la compression et au cryptage tout en réunissant une arborescence de plusieurs fichiers.

Logiciel

Un logiciel est l'ensemble des programmes, procédures et ressources nécessaires au fonctionnement d'un système. Les logiciels sont utilisées par deux types de personnes, les développeurs, se sont ceux qui créent le logiciel mais aussi par l'utilisateur qui utilise et exploite le logiciel. Le logiciel est présent dans tous les systèmes informatiques. Un logiciel est attribué à une tache précise, il y a donc autant de logiciel que de taches à réaliser sur un système.

Pour le démarrage d'un ordinateur il y a plusieurs logiciels exploités : le BIOS pour le démarrage, le système d'exploitation(Linux, Windows, MacOSX, ..) qui est un gros logiciel puis d'autres logiciels pour différentes taches, lire des mails (Thunderbird,..), aller sur internet (Firefox, Chrome,…), ou juste la calculatrice.

Fichier

Un fichier est un ensemble de données regroupées en séquences d'octets, une séquence de 0 et de 1. Il existe plusieurs types de fichier, il y a :

  • Des programmes, contenant des séquences d'ordres compréhensibles par l'ordinateur.
  • Des images, sons, vidéo, traitement de texte, ce sont des données qui sont encodées pour pouvoir ensuite être lu par l'utilisateur à travers un logiciel qui va décoder ces données.
  • Des textes bruts, ils contiennent juste des données textes, des suites de caractères.

C'est différent fichiers sont reconnu par l'ordinateur grâce à une séquence contenues dans les métadonnées du fichier et aussi grâce à son extension. Il existe pleins de types d'extensions , chacune correspondant a un type de fichiers différents. Quelques exemples :

  • *.exe, *.bat pour les fichiers exectuables.
  • *.gif, *.jpeg, *.png pour les fichiers images.
  • *.mp3, *.ogg, *.wav pour les fichiers audios.
  • *.mp4, *.flv, *.avi pour les fichiers vidéos.
  • *.odt, *.docx pour les fichiers de traitement de texte.
  • *.txt, *.log, *.md pour les fichiers de texte brut.
