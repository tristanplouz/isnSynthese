Le web

HTML

Présentation

L’Hypertext Markup Language, généralement abrégé HTML, est le format de données conçu pour représenter les pages web. C’est un langage de balisage permettant d’écrire de l’hypertexte, d’où son nom. HTML permet également de structurer sémantiquement et de mettre en forme le contenu des pages, d’inclure des ressources multimédias dont des images, des formulaires de saisie, et des programmes informatiques. Il permet de créer des documents interopérables avec des équipements très variés de manière conforme aux exigences de l’accessibilité du web.

Il s'agit d'un langage de description qui fonctionne sous la forme de blocs imbriqués les uns dans les autres.

Historique

En 1991, Tim Berners-Lee met au point le langage HTML pour mettre en forme un document, le logiciel qui permet de lire ce fichier et de l'afficher (le navigateur ou browser en anglais) ainsi que le protocole qui permet le transfert des informations.

Aujourd'hui nous fonctionnons avec la version 5 du langage HTML.

Les Bases du HTML

Une page HTML de base comporte un balise <html>. Dans cette balise nous retrouvons une balise définissant l’en-tête <header> et une balise pour le corps de la page <body>

Dans l’entête de la page nous définissons le type d'encodage, les différents import et les autres méta-données.

Dans le corps de la page nous avons le contenu qui sera affiché à l'écran, et structuré grâce à d'autres balises tel que <h1> <p> <article> <section> <div> <blockquote>. Une page HTML ressemble donc à:


<!DOCTYPE html>
<html lang="en">
   <head>
	  <meta charset="utf-8" />
	  <title>Le titre de la page</title>
	  <meta name="generator" content="Geany 1.25" />
	  <link rel="stylesheet" href="mon fichier.css" />
   </head>
   <body>
	  <section>
		 <article>
			 <h1>Un titre important</h1>
			 <p>Un paragraphe</p>
			 <ul>
			Une liste
			   <li>Avec</li>
			   <li>des</li>
			   <li>éléments</li>
			 </ul>
		 </article>
	  </section>
   </body>
</html>

CSS

Présentation

Les feuilles de style en cascade, généralement appelées CSS de l'anglais Cascading Style Sheets, forment un langage informatique qui décrit la présentation des documents HTML et XML. Les standards définissant CSS sont publiés par le World Wide Web Consortium (W3C). Introduit au milieu des années 1990, CSS devient couramment utilisé dans la conception de sites web et bien pris en charge par les navigateurs web dans les années 2000.

Nous utilisons aujourd'hui la version numéro 3 du CSS

les bases du CSS

Le CSS est composé de règle qui agisse sur des éléments. Il y a trois manières d’intégrer du code CSS dans une page web. Seule la dernière méthode est à utilisé.

  • Il s'agit d’intégrer le CSS dans la balise concernée, comme dans l'exemple suivant:

    <p style="color:bleu">

  • Il s'agit de placer le code CSS dans une balise dédier dans l’entête du fichier:

    
<style>
p{
	color:bleu;
}						
</style>

  • Il s'agit de mettre le code CSS dans un fichier externe et de l'inclure

    
<link rel="stylesheet" href="mon fichier.css"/>


    
p{
		color:bleu;
}

Les règles sont définies par un sélecteur(balise, id, class) et des propriétés qui définissent l'aspect graphique.(margin, color, border,...)
