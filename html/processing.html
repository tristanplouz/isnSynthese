Processing

Qu'est ce que Processing

Processing est un environnement de développement en Java développé pour des artistes. Il est conçu pour une utilisation simple et à la portée de tous pour la réalisation de petits programmes graphiques. Pour plus de détails sur Processing et son utilisation, la référence ou le manuel peuvent être utile.

Les bases

La programmation sous Processing repose sur plusieurs règles strictes, à la fin d'une ligne il faut mettre un point virgule ';' et les blocs d'instruction sont entourés d'accolades '{}'.

Les variables

Les variables sont des endroits dans la mémoire où l'on peut stocker des informations de différents types. Processing étant basé sur du Java, il possède tous les types de variables primitives du Java.
On peut donc utiliser les types suivants :

  • int : Il s'agit d'une variable numérique stockant des entiers pouvant aller de -2 147 483 647 à 2 147 483 647
  • float : Il s'agit aussi d'une variable numérique mais elle peut stocker des valeurs décimales de -3.40282347*1038 à 3.40282347*1038 mais avec un manque de précision.
  • double : Il s'agit aussi d'une variable numérique décimale mais elle peut stocker de -3.40282347*1038 à 3.40282347*1038 avec plus de précision qu'un float.
  • boolean : Il s'agit d'une variable logique pouvant uniquement stocker True ou False.
  • Char : Il s'agit d'une variable pouvant stocker un symbole Unicode.

A ces types de variables on peut rajouter d'autres types tel que :

  • Array : Il s'agit d'un tableau de variables, il peut être à plusieurs dimensions.
  • String : Il s'agit d'une chaîne de caractères, un tableau de char. Cela permet de stocker du texte.

Les fonctions

Pour fonctionner correctement, un programme Processing doit contenir au moins deux fonctions, le void setup et le void loop. La première de ces fonctions, le void setup , est exécutée une seule fois, à l'initialisation du programme. La seconde, le void loop, sera exécutée à chaque mise à jour de l’écran, c'est à dire environs 30 fois par seconde. Ensuite pour pouvoir faire des choses, on a notre disposition plein de fonctions différentes. Nous allons voir quelques fonctions basiques :

  • size() : Cette fonction permet de choisir la taille de la fenêtre affichée par Processing.
  • background() : Cette fonction sert à définir la couleur du fond de la fenêtre de notre programme.
  • ellipse() : Cette fonction permet de dessiner une ellipse dans notre fenêtre. On déclare l'ellipse grâce aux coordonnées et à sa largeur et longueur.
  • rect() : Cette fonction permet de dessiner un rectangle dans notre fenêtre. On lui place en paramètre les même choses que pour l'ellipse.
  • fill() : Cette fonction est utilisée pour définir la couleur de ce que l'on va dessiner après.
  • Text() :Cette fonction permet d’écrire une String sur l’écran à une position précise. On peut rajouter des paramètres sur le texte avec les fonctions textSize(), textAlign(), textFont().

Toutes les fonctions sont répertoriées et documentées dans la référence. Nous pouvons aussi créer nos propres fonctions pour faire plusieurs fois la même séquence sans avoir à tous réécrire. Nous allons donc déclarer notre fonction en commençant par le type de fonction, si la fonction renvoi quelque chose à la fin de son exécution, il faut mettre le type de variable que l'on retourne, si la fonction ne retourne aucune information, le type de la fonction sera void. On écrit ensuite le nom de notre fonction. On le suit de parenthèse on l'on peut mettre des arguments pour la fonction. On forme ensuite un bloc avec les accolades pour entouré le code exécuté par la fonction.

Les structures conditionnelles

Les structures de contrôle sont la base de la programmation, elles permettent de relier différentes informations pour exécuter d'autres tâches.

Les conditions

Les conditions permettent de comparer deux valeurs pour ensuite réaliser des tâches si les conditions sont remplis ou non. Il y a trois mot clé différents essentiels pour les conditions if, else et else if. Le premier permet de comparer deux valeurs grâce à des opérations logiques ( ''=='' égale, ''!='' différent, ''<'' inférieur, ''>'' supérieur, …), si la condition est remplie, cela exécute la ou les tâches. Au lieu de réaliser plusieurs conditions, on peut lui demander de comparer plusieurs choses avec les opérateurs ''&&'' et ''||'', qui signifient ''ET'' et ''OU''. Else permet de rajouter un ''sinon'', si la condition précédente n'est pas remplie, il exécute la ou les tâches du else. Else if est un mélange des deux, si la condition précédente n'est pas remplie mais que cette condition est rempli alors il exécute la ou les tâches. Lorsqu'on a plusieurs conditions à la suite, où l'on compare une valeur à plusieurs autres, il est préférable d'utiliser la structure switch case que de faire suivre plusieurs if, else if, else.

Les boucles

Les boucles sont des structures permettant de répéter plusieurs fois certaines tâches, en fonction d'un incrément ou d'une condition. Les mots clés sont for, pour un nombre de boucle précis, ou while, pour répéter tant que la condition donnée est remplie.
