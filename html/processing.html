Processing

Qu'est ce que Processing

Processing est un environnement de développement en Java développé pour des artistes. Il est conçu pour une utilisation simple et à la portée de tous pour la réalisation de petits programmes graphiques. Pour plus de détails sur Processing et son utilisation, la référence ou le manuel peuvent être utile.

Les bases

La programmation sous Processing repose sur plusieurs règles strictes, à la fin d'une ligne il faut mettre un point virgule ';' et les blocs d'instruction sont entourés d'accolades '{}'.

Les variables

Les variables sont des endroits dans la mémoire où l'on peut stocker des informations de différents types. Processing étant basé sur du Java, il possède tous les types de variables primitives du Java.
On peut donc utiliser les types suivants :

  • int : Il s'agit d'une variable numérique stockant des entiers pouvant aller de -2 147 483 647 à 2 147 483 647
  • float : Il s'agit aussi d'une variable numérique mais elle peut stocker des valeurs décimales de -3.40282347*1038 à 3.40282347*1038 mais avec un manque de précision.
  • double : Il s'agit aussi d'une variable numérique décimale mais elle peut stocker de -3.40282347*1038 à 3.40282347*1038 avec plus de précision qu'un float.
  • boolean : Il s'agit d'une variable logique pouvant uniquement stocker True ou False.
  • Char : Il s'agit d'une variable pouvant stocker un symbole Unicode.
